# AndroidQuizApp
### 基於線上課內容練習製作Quiz app
更換主題、題目內容、UI更換、ICON更換設計、發布APK

#### 運用概念
- android UI widget,clickListener設計運用
- constraint layout,linear layout,scroll view等部局運用
- 多activity頁面連結、資料傳遞(intent)
- 利用data class裝載問題資料
- 判斷式及邏輯運算子
